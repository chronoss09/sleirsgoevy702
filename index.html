    If the system crashes (looks like instant powerdown), press the power button on the PS4 (NOT on the gamepad) until it turns on again, then retry.
  2. If you get a "There is not enough free system memory" message without any pop-ups, then it means that the webkit exploit has failed. Click OK and retry.
  3. For advanced users: To load your own payloads using NetCat, run NETCAT and send the payload to TCP port 9021 (9020 for the legacy version).
